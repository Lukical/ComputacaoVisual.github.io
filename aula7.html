APIs Gráficas

As APIs gráficas são responsáveis pelo processamento gráfico 3D, fazendo a cominicação entre os drivers e a placa de vídeo. A API fica em uma camada entre o driver e a engine de motor gráfico, onde o driver é responsável por levar instruções para o hardware comunicar entre os aplicativos utilizados no computador.

As APIs atualmente mais utilizadas atualmente são o DirectX e Vulkan, onde são muito utilizadas para o processamento de gráfico de aplicações.
